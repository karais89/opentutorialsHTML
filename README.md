## HTML 기술 소개

HyperText = 하이퍼텍스트를 가장 중요한 특징으로 하는

Markup = 마크업이라는 형식을 가진

Language = 컴퓨터 프로그래밍 

언어의 핵심은 **약속** 사람과 컴퓨터 사이의 약속. 사람과 웹브라우저의 약속.
